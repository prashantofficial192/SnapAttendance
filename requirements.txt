Modules and Libraries 

1. Flask (3.0.0):

=> Flask is a lightweight WSGI web application framework in Python.
=> It provides tools, libraries, and technologies for building web applications.


2. Flask-Cors (4.0.0):

=> Flask extension for handling Cross-Origin Resource Sharing (CORS), allowing cross-domain requests.


3. Flask-SocketIO (5.3.6):

=> Flask extension for handling WebSocket connections.


4. Jinja2 (3.1.2):

=> A template engine for Python, used in Flask for rendering HTML templates.


5. numpy (1.26.2):

=> A fundamental package for scientific computing with Python.
=> Used for numerical operations and array manipulation.


6. opencv-python (4.9.0.80):

=> OpenCV (Open Source Computer Vision Library) is used for computer vision tasks, such as image and video processing.
=> opencv-python is the Python wrapper for the OpenCV library.


7. plyer (2.1.0):

=> A Python library for accessing features commonly found in mobile platforms.
=> Used for sending notifications in this project.


8. python-dateutil (2.8.2):

=> Provides extensions to the standard datetime module in Python.
=> Used for working with dates and times.


9. python-engineio (4.8.2):

=> A Python implementation of the Engine.IO protocol.
=> Used in conjunction with Flask-SocketIO for WebSocket connections.


10. python-socketio (5.11.0):

=> A Python implementation of the Socket.IO protocol.
=> Used for bidirectional communication between the web client and the server.


11. SQLAlchemy (2.0.25):

=> SQLAlchemy is a SQL toolkit and Object-Relational Mapping (ORM) library for Python.
=> It provides a way to interact with relational databases using Python objects.


12. wheel (0.29.0):

=> A built-package format for Python.
=> It is used for distributing packages.


13. yarl (1.9.3):

=> Yet another URL library for Python.
=> Used for working with URLs.


14. websockets (12.0):

=> A library for building WebSocket servers and clients in Python.


15. face-recognition (1.3.0):

=> A simple face recognition library for Python.
=> Used for face recognition tasks in this project.


16. face_recognition_models (0.3.0):

=> Pre-trained models for face recognition tasks used by the face-recognition library.
